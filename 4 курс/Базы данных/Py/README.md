# Загрузка JSON в Access (Дом пристарелых)

## Что нужно

1. **Python 3** с установленным пакетом `pyodbc`:
   ```bash
   pip install -r requirements.txt
   ```
2. **Таблицы в базе Access** — созданы по скрипту `create_nursing_home_db.sql` (в папке выше).
3. **Драйвер ODBC для Access** на Windows:
   - часто уже есть вместе с Office (Microsoft Access Driver или Microsoft ACE ODBC Driver);
   - если нет — установите [Microsoft Access Database Engine](https://www.microsoft.com/en-us/download/details.aspx?id=54920) (выберите разрядность под ваш Python: 32-bit или 64-bit).

## Как запустить

Если путь к папке содержит русские буквы (например, «Рабочий стол», «Учеба», «Базы данных»), при запуске по полному пути к файлу путь может искажаться и появится ошибка *No such file or directory*. Используйте один из способов:

1. **Двойной щелчок по `run.bat`** в этой папке — скрипт запустится из своей директории без полного пути.
2. **Из терминала Cursor** (папка проекта уже открыта): откройте терминал и выполните:
   ```bash
   python json_to_access.py
   ```
   Рабочая папка должна быть `Py` (Cursor обычно открывает её как корень проекта).
3. **Из cmd/PowerShell**: перейдите в папку `Py`, затем запустите:
   ```bash
   cd "E:\Рабочий стол\Учеба\Study\4 курс\Базы данных\Py"
   python json_to_access.py
   ```

## Как пользоваться

1. После запуска программы:
2. Нажмите **«Выбрать JSON»** и укажите файл с данными (например, `nursing_home_data.json`).
3. Нажмите **«Выбрать базу»** и укажите файл базы Access (например, `Dom1.accdb`).
4. Нажмите **«Загрузить данные в базу»**.

Данные будут вставлены в таблицы в правильном порядке (сначала справочники, затем Employee, Resident, затем процедуры, платежи и посещения). Поле `id` (COUNTER) заполняется базой автоматически.
